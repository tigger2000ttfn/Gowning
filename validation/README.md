# Validation Package - MATC Gowning Qualification System

Computer System Validation (CSV) package for GQS, versioned with the code. As a custom-developed
application, GQS is GAMP 5 (highest rigor). Drafted in a generic structure and refit to Astellas
templates and numbering when provided. QA owns review and approval.

Planned contents: Validation Plan, URS, Functional Spec, Design Spec, Risk Assessment (incl.
Part 11 assessment), Requirements Traceability Matrix, IQ, OQ, PQ, Validation Summary Report,
and supporting SOPs (access management, backup/restore, audit-trail review, electronic
signatures, change control).
